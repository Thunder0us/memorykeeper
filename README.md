## Как запустить
Просто скачайте .exe из релиза 1.1 и запустите файл.

Запустить можно и через команду python memory_keeper.py или же через MemoryKeeper.exe

Присутствуют баги с отображением, поэтому в случае:
1. Не удаления пунктов
2. Не добавления пунктов и напоминаний

Требуется просто перезапустить приложение. 

Также в данный момент через запуск через .exe не работают уведомления Windows, чтобы всплывало упоминание по таймеру. В случае если требуется именно такой функционал - нужно запустить приложение через команду
