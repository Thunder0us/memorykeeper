!!!!!
Выкладываю именно файл, а не формат .exe, так как после сборки файла - перестают работать ключевые функции приложения, однако в случае если требуется именно файл в .exe формате, то нужно выполнить следующую команду в Powershell с указанием папки с файлом, как итогового.

flet pack memory_keeper.py --name "Memory Keeper" --product-version 1.0 --file-version 1.0
!!!!!

Запустить можно и через команду python memory_keeper.py или же через папку dist.

Присутствуют баги с отображением, поэтому в случае:
1. Не удаления пунктов
2. Не добавления пунктов и напоминаний

Требуется просто перезапустить приложение. 

Также в данный момент через запуск через .exe не работают уведомления Windows, чтобы всплывало упоминание по таймеру. В случае если требуется именно такой функционал - нужно запустить приложение через команду
